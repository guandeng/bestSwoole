#### bestswoole
> bestswoole基于Swoole开发，最大限度保留Swoole原有特性

#### 安装
> 安装代码 composer require guandeng/bestswoole:dev-master

> 运行脚手架工具安装框架 php vendor/guandeng/bestswoole/src/Install.php

> 到bin目录下  php swoole_server.php start

#### 主要特性
- 1、MVC架构
- 2、支持异步/同步/协程开发
- 3、事件驱动异步编程模式
